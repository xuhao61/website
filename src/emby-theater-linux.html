Emby Theater for Linux

What you want to watch – right up front

Designed with with typical usage in mind, you won’t have to go hunting through your library to find the content you are most likely wanting to watch right now. The home screen shows you the items you were watching but hadn’t finished, your latest un-watched movies, the next episode of all your current TV shows and what’s on Live TV right now.

Debian / Ubuntu / Linux Mint

Download one of the following packages based on your system architecture:
  • X64
  • Arm64 (aarch64)
  • Armv7 (armhf)
Install Steps
sudo apt-get update
sudo apt-get install cec-utils libasound2 libatomic1 libc6 libegl1 libgcc-s1 libpulse0 libstdc++6
sudo dpkg -i /path/to/deb
Run Emby Theater by launching from your desktop applications list.

Arch Linux

Run the following:

pacman -S emby-theater

Run Emby Theater by launching from your desktop applications list.

CentOS

Install one of the following based on your system architecture:
  • X64: yum install https://github.com/MediaBrowser/emby-theater-electron/releases/download/3.0.19/emby-theater-rpm_3.0.19_x86_64.rpm
  • Arm64 (aarch64): yum install https://github.com/MediaBrowser/emby-theater-electron/releases/download/3.0.19/emby-theater-rpm_3.0.19_aarch64.rpm
