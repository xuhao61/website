ASUSTOR
Installation
– Install Emby Server from App Central on the ASUSTOR NAS web page
– Install Mono and X.org using App Central
– Start the server from the ASUSTOR web page
Run Setup Wizard

Visit http://localhost:8096

FreeNAS

– Install the MediaBrowser plugin using the FreeNAS web interface

– Start the server using the FreeNAS web interface

– You may need to configure storage within your FreeNAS Jail in order to allow Emby to access it. See Adding storage to FreeNAS.

More Information
Open Media Vault
– Install the omv-extras plugin
– In the secondary tab, activate the mono repository
– Install the omv-mediabrowser plugin from the plugins page
– Activate the mediabrowser service from the plugins page
– Click the Administration button to launch the Emby startup wizard
More Information
QNAP

Installation
Install Emby Server for QNAP by downloading from the QNAP Community..

Run Setup Wizard

Start Emby from the QNAP web interface, then visit http://localhost:8096 to run the Emby setup wizard.

Updating Emby Server

To get faster access to updates, you can always download the latest version from the QNAP Community.
