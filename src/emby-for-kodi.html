Emby Add-ons for Kodi

Install the EmbyCon Add-on

Just install the EmbyCon Add-on and you can connect directly to your Emby Server. EmbyCon lets you easily browse and play your Emby library using Kodi.

For more information, including a YouTube demonstration, visit the Emby community.

Or Use the Emby for Kodi Sync Add-On

The Emby for Kodi sync add-on has a richer feature set than EmbyCon, but requires a periodic syncing process to download data from your Emby Server. This syncing process can take a little while and may result in increased CPU and memory consumption on your Emby Server.

Installation instructions can be found on our Emby for Kodi Forum.
